# Screenplay Patterns Library

This directory contains reusable patterns for screenplay development.

## Pattern Categories

### Story Structure Patterns
- Three-Act Structure
- Save the Cat! Beat Sheet
- Hero's Journey
- Sequence Method

### Scene Patterns
- Confrontation Scene
- Discovery Scene
- Action Sequence
- Montage
- Flashback/Flashforward
- Phone Conversation (Intercut)

### Dialogue Patterns
- Exposition Through Conflict
- Subtext Techniques
- Dual Dialogue
- Voice Differentiation

### Character Patterns
- Character Introduction
- Arc Transformation Moments
- Relationship Dynamics

## Using Patterns

### Pattern Discovery
Before writing new scenes, search existing patterns:
1. Check this directory for applicable patterns
2. Search existing screenplay for similar scenes
3. Adapt patterns to current needs

### Pattern Creation
When developing new patterns:
1. Document the pattern clearly
2. Provide examples
3. Note when to use/not use
4. Add to appropriate category

## Pattern Template

```markdown
# Pattern: [Name]

## When to Use
- [Situation 1]
- [Situation 2]

## Structure
[Description of the pattern structure]

## Example
[Fountain example]

## Variations
- [Variation 1]
- [Variation 2]

## Common Mistakes
- [Mistake to avoid]
```
