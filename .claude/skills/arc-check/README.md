## Quick Start

This skill activates automatically when you:

- Track character development through a story
- Ensure meaningful character transformation
- Validate emotional journeys
- Identify flat or static characters

## What This Skill Does

Validates character arc development throughout a screenplay covering positive,
negative, transformational, and flat arc types with examples from classic films
like Tootsie, The Godfather, and Empire of the Sun.
