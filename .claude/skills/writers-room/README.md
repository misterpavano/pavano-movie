## Quick Start

This skill activates automatically when you:

- Start a new screenplay
- Plan major rewrites
- Reimagine existing material
- Seek diverse expert perspectives

## What This Skill Does

Convenes a 6-agent collaborative pre-production session where Story Architect,
Dialogue Writer, Scene Writer, Research Specialist, Standards Reviewer, and
Continuity Editor pitch their creative visions before writing begins.
