# Submission Tracker

Track every query, submission, and contest entry here. The industry runs on follow-up and timing. If it's not tracked, it didn't happen.

## How to Use This

- Add an entry the day you submit — don't catch up later
- Update Status as things change
- Note the exact date of contact and any specifics from the response (even a pass is data)
- **Follow-up rule:** One follow-up after 4-6 weeks of silence. If no response after the follow-up, mark as No Response and move on.
- Use the Script Version column — always know which draft you sent where
